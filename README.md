# closewithq.nvim

Just a `FileType` autocmd snippet with builtin patterns, source code from [LazyVim](https://github.com/LazyVim/LazyVim/blob/main/lua/lazyvim/config/autocmds.lua).

## Usage

```lua
-- lazy.nvim
return {
  "2nthony/closewithq.nvim",
  event = "VeryLazy",
  opts = {
    patterns = {
      -- filetypes
    },
  },
}
```
